Detailed Description

The BObject class is the base class of all classes in the Bengine.

The BObject class is a lightweight base class for all classes that belong the the engine. All BObject subclasses are prefixed with a 'B', just like all Qt classes are prefixed with a 'Q'.

Subclassing BObject

In order to subclass BObject, you must include the B_OBJECT macro somewhere in the class declaration:

#include <core/bobject.h>

class SubClass : public BObject
{
     B_OBJECT(SubClass)

public:
     SubClass();
     ~SubClass();
};

This macro does some background magic which will allow the engine to supply things like static type info (see the BClass documentation) through the getStaticClass() function.

Macro Documentation

B_OBJECT( className)

Include this macro in your class definition when subclassing BObject. To use this macro, include the name of your class in the className parameter, as such:

#include <core/bobject.h>

class SubClass : public BObject
{
     B_OBJECT(SubClass)

public:
     SubClass();
     ~SubClass();
};
